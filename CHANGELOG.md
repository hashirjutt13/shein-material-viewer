# Changelog

All notable changes to `shein-material-viewer` will be documented in this file.

## 0.1.0 - 2026-06-08

- Added material-only viewer mode as the default product-card behavior.
- Kept optional material filter mode with rules, presets, dim/hide/badge display, and match exports.
- Added material parsing for clothing and accessory materials including TPU, PC, PU leather, silicone, tempered glass, plastic, metal, zinc alloy, aluminum, resin, and rubber.
- Added manual scan, slow auto scan, delay, and randomness controls.
- Added local caching for successful material scans.
- Added CSV/JSON export for scanned products and optional matches.
- Added parser/settings tests.
- Added full repository documentation.
